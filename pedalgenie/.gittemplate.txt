# Body(아래에 작성)

# Footer(아래에 작성)

# Feat: 카카오 간편 로그인 기능 추가
# 
# - 회원가입 UX 개선을 위해 카카오 간편 로그인 기능 추가

# Fixes: #47 
# Related to: #32, #21

# 제목 첫 글자는 대문자로 작성
# 제목은 명령문으로 작성
# 제목 끝에 마침표(.) 금지
# 제목, 본문, 꼬릿말 사이는 한 줄씩 띄워 분리하기
# 본문은 "어떻게" 보다 "무엇을", "왜"를 설명
# ------ Type ------
# Init      :	프로젝트 초기 생성
# Feat      :	새로운 기능 추가 및 수정
# Fix       :	버그 수정(의도하지 않았던 문제 해결)
# Refactor  :	코드 리팩토링(변수명 수정 등)
# Style     :	코드 스타일 변경(오탈자, 포맷팅, 단순 스타일(색상) 및 에셋 변경 등 기능적 변경 없는 경우)
# Chore     :	프로덕션 코드 외 변경사항(빌드 스크립트, 패키지 매니저 등)
# Docs      :	문서 수정
# Comment   :	주석 추가 및 변경
# Test      :	테스트 코드, 리팩토링 테스트 코드 추가 및 변경
# Move      :	파일 및 폴더 이동
# Rename    :	파일명 및 폴더명 수정
# Remove    :	파일 및 폴더 삭제
# ------ Footer ------
# Fixes       : 이슈 수정중 (아직 해결되지 않은 경우)
# Resolves    : 이슈 해결했을 때 사용
# Ref         : 참고할 이슈가 있을 때 사용
# Related to  : 해당 커밋에 관련된 이슈번호 (아직 해결되지 않은 경우)
